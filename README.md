fip
===

fip - Front-end Image Processor - 前端图像处理器

## 使用

需要开启HTTP服务器。
在Windows下且安装了Python时，可以执行 `run-py2.bat` 或 `run-py3.bat` 。

## 包含的组件

- fip.js                图像处理核心
- fip.find-padding.js   检测边距算法
- fip.find-gap.js       检测竖栏间距算法
- fip.pick-anchor       检测鼠标附近定位点算法

## 使用的地方

- Perpetura & EDU 切图出卷功能
- Perpetura 定位点标注页
